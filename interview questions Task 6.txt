A template in Flask is an HTML file that contains dynamic placeholders using Jinja2 syntax. Flask fills those placeholders with actual data before sending the page to the browser.

Use render_template() and pass variables like key-value pairs:

4Ô∏è.How are forms handled in Flask?

Define a route that supports GET + POST

Fetch form values using request.form

Example:

from flask import request

@app.route("/submit", methods=["GET", "POST"])
def submit():
    if request.method == "POST":
        name = request.form["name"]
        return f"Hello {name}"
    return render_template("form.html")

5Ô∏è.What is render_template()?

7Ô∏è.POST vs GET method in forms?
Feature	GET	POST
Visibility	Data shown in URL	Data hidden in body
Usage	Search, navigation	Login, sensitive forms
Data Length	Limited	Not limited
Security	Less secure	More secure
